INSTRUÇÕES DE INSTALAÇÃO - FLORICULTURA TAUBATÉ

1. Extraia este arquivo ZIP.
2. Você verá uma pasta chamada 'landing-page' (ou os arquivos soltos, dependendo de como extraiu).
3. Abra o arquivo 'index.html' no seu navegador para testar.
4. Para colocar no ar (Hostinger ou similar):
   - Acesse o Gerenciador de Arquivos ou FTP.
   - Vá para a pasta 'public_html'.
   - Faça o upload dos arquivos:
     - index.html
     - style.css
     - script.js
     - pasta 'assets'
   - Certifique-se de que o 'index.html' esteja na raiz do seu domínio (dentro de public_html).

Dúvidas? Entre em contato com o suporte.
